# 🤖 Rule-Based AI Chatbot

## 📌 Project Overview
This project is a simple Rule-Based AI Chatbot developed using Python as part of the DecodeLabs Artificial Intelligence Internship.

The chatbot interacts with users by responding to predefined inputs using `if-elif-else` statements and runs continuously until the user exits.

---

## 🚀 Features
- Greets users
- Responds to simple predefined questions
- Handles exit commands (`bye` and `exit`)
- Uses `if-elif-else` decision-making logic
- Runs in a continuous loop

---

## ▶️ How to Run


2. Navigate to the project folder:
```bash
cd AI-Rule-Based-Chatbot
```

3. Run the chatbot:
```bash

```
